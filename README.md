# Machine Learning Portfolio

Welcome to my **Machine Learning Learning & Research Portfolio**.

This repository documents my journey of understanding machine learning concepts from the ground up through **hands-on coding, real-world datasets, exercises, and research studies**.

The goal of this repository is to build strong conceptual and practical knowledge in machine learning by combining:

• Conceptual understanding  
• Mathematical intuition  
• Practical implementation  
• Real-world applications  
• Research exploration

# Repository Structure

This repository is organized into structured learning modules.

## 1. Regularization in Linear Models Using Real-World Data
This project explores techniques used to prevent overfitting in machine learning models.

Topics covered:
- Ridge Regression
- Lasso Regression
- Bias-Variance Tradeoff
- Model Generalization

Dataset:
Real-world dataset used to demonstrate how regularization improves model performance.

Focus:
Understanding how penalties applied to model coefficients help control complexity.

## 2. Understanding Linear and Polynomial Regression Using Synthetic Data

Topics covered:

- Underfitting vs Overfitting
- Visualization of regression curves

Focus:
Building intuition on how models learn patterns from data.
